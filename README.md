
# swagger-extractor
An simple swagger OpenApi Documentation extractor

## How to Use

**Type :**
`npm run extract`  to extract all OpenApi documentation generated by Swagger to this model:

     [ {
             "url": "An url",
             "method": "A http method",
             "tags": "Swagger tag",
             "description": "Swagger description",
             "operation": "java method name"
         } ]
